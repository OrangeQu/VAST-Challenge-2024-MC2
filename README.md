## 二、数据预处理 v3 (`preprocess_v3.py`)

### 2.1 核心改进
相比 v2 版本，v3 主要做了以下优化：

| 改进项 | v2 版本 | v3 版本 |
| :--- | :--- | :--- |
| **数据归一化** | StandardScaler (均值-标准差) | **MinMaxScaler** (0-1归一化) |
| **降维算法** | PCA 近似 | **真实 t-SNE** 非线性降维 |
| **特征维度** | 11 个特征 | **12 个特征**（新增捕鱼强度、入侵频率） |
| **输出文件** | `processed_data_v2.json` | `processed_data_v3.json` (独立文件) |

### 2.2 提取的行为特征
共提取 **12 个行为特征**，用于船舶行为分析：

| 序号 | 特征名 | 说明 | 计算方式 |
| :---: | :--- | :--- | :--- |
| 1 | `night_ratio` | 夜间活动率 | 夜间访问次数 / 总访问次数 |
| 2 | `night_fishing_ratio` | 夜间捕鱼率 | 夜间渔场访问 / 夜间总访问 |
| 3 | `fishing_ground_visits` | 渔场访问次数 | 累计进入渔场的次数 |
| 4 | `avg_dwell_hours` | 平均停留时间 | 所有停留时长的平均值 |
| 5 | `entropy` | 移动熵 | 位置分布的信息熵（多样性） |
| 6 | `protected_dwell_ratio` | 保护区停留比例 | 保护区内停留时长 / 总停留 |
| 7 | `fishing_intensity` | 捕鱼强度 | 渔场访问次数 / 活跃天数 |
| 8 | `intrusion_frequency` | 入侵频率 | 保护区访问次数 / 活跃天数 |
| 9 | `locations_per_day` | 日移动频率 | 访问地点数 / 活跃天数 |
| 10 | `avg_periodicity` | 周期性 | 访问间隔的规律性评分 |
| 11 | `total_pings` | 总Ping数 | 累计移动信号总数 |
| 12 | `unique_locations` | 地点多样性 | 访问过的不同地点数量 |

### 2.3 t-SNE 参数配置
```python
tsne = TSNE(
    n_components=2,      # 降维到 2D
    perplexity=15,       # 适合局部群体挖掘
    random_state=42,     # 可复现结果
    max_iter=1000,       # 迭代次数
    init='pca'           # PCA 初始化
)
```

## 三、Q3 可视化设计

### 3.1 页面布局
采用**单列布局**，从上到下依次展示：
* **分析摘要** - 定性结论概述
* **统计栏** - 关键指标卡片
* **t-SNE 行为空间地图** - 核心可视化，带交互功能
* **雷达图** - 行为指纹对比
* **平行坐标图** - 多维特征筛选
* **高度可疑船舶清单** - 详细数据表格

---

### 3.2 图表详解

#### 3.2.1 t-SNE 行为空间聚类地图
| 属性 | 说明与作用 |
| :--- | :--- |
| **作用** | 将 12 维行为特征映射到 2D 空间，直观展示船舶聚类形态 |
| **颜色编码** | 🔴 红色 = SouthSeafood / 🟣 紫色 = 高度可疑 (≥95%) / ⚪ 灰色 = 普通渔船 |
| **交互功能** | 悬停提示、点击选中、缩放平移、框选区域、高亮按钮 |
| **数据来源** | `vessels[].tsne_x`, `vessels[].tsne_y` |

**交互控制面板：**
* 🔄 **重置视图** - 恢复原始视角
* 🔍 **放大/缩小** - 步进式缩放
* ⭐ **高亮 SouthSeafood** - 临时放大红色点
* ⚠️ **高亮可疑船舶** - 临时放大紫色点

#### 3.2.2 行为指纹雷达图
| 属性 | 说明与作用 |
| :--- | :--- |
| **作用** | 多维度对比行为特征轮廓 |
| **对比对象** | SouthSeafood (红线) vs Top 3 相似 (紫虚线) vs 船队平均 (蓝区域) |
| **维度** | 8 个核心行为特征 |
| **数据来源** | `ssVessels` 平均值、`highSuspectVessels` Top 3 平均值、船队平均值 |

#### 3.2.3 平行坐标图
| 属性 | 说明与作用 |
| :--- | :--- |
| **作用** | 展示各船舶在多维特征上的表现 |
| **线条颜色** | 🔴 红色 = SouthSeafood / 🟣 紫色 = 可疑船舶 / ⚪ 灰色 = 其他船只 |
| **维度** | 10 个行为特征 |
| **交互支持** | 轴缩放（轴上拉丝筛选） |

#### 3.2.4 高度可疑船舶清单
| 属性 | 说明与作用 |
| :--- | :--- |
| **筛选条件** | 相似度 ≥ 95% |
| **显示字段** | 船名、相似度、公司、夜间活动、夜间捕鱼、停留时间、保护区停留、捕鱼强度、入侵频率 |
| **数据来源** | `similar_vessels` 中筛选 `score >= 0.95` |
